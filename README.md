# 数学ゲームコレクション

Tkinter（Python標準のGUIライブラリ）で作成した、数学をテーマにしたミニゲーム集です。
3種類のゲームをひとつのメニュー画面から選んで遊べます。

## ゲーム内容

### 1. 素因数分解ゲーム
ランダムに出題される数字を、カンマ区切りで素因数分解して答えます。
「かんたん」「ふつう」「むずかしい」の3段階の難易度を選べます。
ヒント機能で最小の素因数を確認することもできます。

### 2. 素数判定クイズ
出題された数字が「素数」か「素数でない」かをボタンで判定します。
こちらも難易度（出題範囲）を選択できます。

### 3. Make10パズル
ランダムに与えられる1〜9の数字4つと、四則演算（+ - * /）・括弧を使って、
計算結果が10になる式を作ります。

いずれのゲームも、名前と問題数（3問・5問・10問）を選んでスタートします。
全問終了後に結果が表示され、「もう一度遊ぶ」「スタート画面に戻る」「トップに戻る」を選べます。

## 必要環境

- Python 3.x（tkinterは標準ライブラリのため追加インストール不要）
- Windows / Mac / Linux いずれでも動作します

事前に [Python公式サイト](https://www.python.org/downloads/) からPythonをインストールしてください。
インストール時に「Add Python to PATH」にチェックを入れることをおすすめします。

## 使い方

### 1. リポジトリをダウンロード

```bash
git clone https://github.com/【ユーザー名】/【リポジトリ名】.git
cd 【リポジトリ名】
```

Gitを使わない場合は、GitHubページ右上の「Code」→「Download ZIP」からダウンロードして解凍してください。

### 2. ゲームを起動

**Windowsの場合：**
`start.bat` をダブルクリックするだけで起動します。

**Mac / Linuxの場合、またはコマンドで起動したい場合：**
```bash
python game_top.py
```
（環境によっては `python3 game_top.py` が必要な場合があります）

### 3. 遊び方

起動するとメニュー画面が表示されるので、遊びたいゲームのボタンを押してください。
各ゲーム画面の「遊び方」ボタン（Make10パズルのみ）またはラベルの説明文を確認しながら進められます。
